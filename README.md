# Socratic Questioning for Agile Coaches

An interactive, self-contained web page that helps agile coaches ask better questions in their work with product teams.

## What's in it

- **6 Socratic Question Types** — expandable cards with agile-specific examples, timing guidance, and coaching tips
- **Ceremony Quick-Filter** — select Sprint Planning, Refinement, Review, Retro, Daily, or 1-on-1 to see relevant questions across all types
- **Circular Question Generator** — pick two team roles and get contextual questions that surface hidden team dynamics
- **Cognitive Restructuring Worksheet** — interactive fillable form for 1-on-1 coaching sessions
- **Practical Flow Guide** — visual conversation flow mapping question types to coaching phases

## Usage

Open `index.html` in a browser. No build step, no dependencies, works offline.

## Background

Inspired by:
- The six types of Socratic questions (conceptual clarification, probing assumptions, probing rationale/evidence, questioning viewpoints, probing implications, questions about the question)
- Cognitive restructuring techniques from therapeutic coaching
- Circular/systemic questioning from team coaching research
- Agile coaching practices for product teams
