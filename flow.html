PhaseQuestion TypeWhat You're DoingExample Moment
OpeningClarificationSurfacing what the team is actually thinking"What's actually on your mind about this sprint?"
DeepeningAssumptionsShaking the foundations of unexamined beliefs"What are we taking for granted here?"
GroundingEvidenceMoving from opinion to data"What data do we have? What don't we know?"
WideningPerspectivesBreaking tunnel vision, building collective intelligence"How would [other role / user / team] see this?"
Forward-lookingImplicationsConnecting decisions to consequences"If we do X, then what happens?"
ReflectingMetaBuilding self-awareness about the thinking process"What made this conversation useful?"
The key skill: Not following this as a script, but listening for which type of question the team needs right now. If they're vague → clarify. If they're overconfident → probe assumptions. If they only see one angle → widen. If they're avoiding the real issue → go meta.
